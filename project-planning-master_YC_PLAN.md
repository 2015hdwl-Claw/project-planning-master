## 專案概述

### 產品定位
**Project Planning Master（專案規劃大師）** 是一個自動化的專案規劃管理系統，標準化所有專案的 YC 5-phase 開發流程，確保每個專案都遵循最佳實踐。

### 核心價值
- **自動化文檔生成**: 一鍵建立標準化專案結構
- **強制執行規範**: 確保所有專案遵循相同命名和結構標準
- **進度可視化**: 即時追蹤所有專案的階段和完成度
- **品質保證**: 強制執行 Code Review、TDD 等最佳實踐

### 商業模式
- **開源工具**: 免費提供給開發社群使用
- **企業版本**: 未來可提供團隊協作和雲端同步功能

- **特色**:
  - 多專案管理
  - 階段完成度追蹤
  - 里程碑提醒

### 4. YC 5-phase 流程管理
- **功能**: 引導完成標準開發流程
- **特色**:
  - Phase 檢核表
  - Skills 推薦
